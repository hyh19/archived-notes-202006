## Installation

安装依赖：JDK 1.7+

<http://dl.mycat.io/>

```bash
# CentOS 7.3
curl -L https://gitee.com/mrhuangyuhui/notes/raw/master/scripts/install-mycat-bin.sh | bash

$ mycat start
Starting Mycat-server...
$ mycat status
Mycat-server is running (3162).
$ mycat stop
Stopping Mycat-server...
Mycat-server was not running.
```
