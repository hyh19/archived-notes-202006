## Install

<https://guides.cocoapods.org/using/getting-started.html>

```bash
sudo gem install cocoapods
```

遇到 Ruby 或 Gem 的问题执行以下命令安装

```bash
sudo gem update --system
sudo gem install cocoapods -n /usr/local/bin
```

桌面客户端 <https://cocoapods.org/app>

## Commands

查看帮助

```bash
pod --help
```

查看版本

```bash
pod --version
```

```bash
# 生成 Podfile 文件
pod init
```

```bash
# 安装依赖库
pod install
```
