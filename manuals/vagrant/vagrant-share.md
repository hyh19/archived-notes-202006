# Vagrant Share

<https://www.vagrantup.com/docs/share/>

## Install

<https://www.vagrantup.com/docs/share/#installation>

> 注意：需要先安装 [ngrok](https://ngrok.com/download)

```bash
# Mac
mv ~/Downloads/ngrok /usr/local/bin/
```

```bash
vagrant plugin install vagrant-share
```

## Commands

```bash
vagrant share
```
