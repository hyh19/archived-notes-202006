# Wget

查看帮助

```bash
wget -h
wget --help
```

下载文件到指定目录

```bash
wget https://download.virtualbox.org/virtualbox/rpm/el/virtualbox.repo -P /etc/yum.repos.d/
```

```bash

Logging and input file:
  -q,  --quiet                     quiet (no output)

Download:
  -O,  --output-document=FILE      write documents to FILE
```
