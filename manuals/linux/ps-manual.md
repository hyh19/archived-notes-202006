# PS Manual

## Commands

查看帮助信息

```bash
ps --help

# 查看使用手册
man ps
```

列出当前 `bash` 环境下的进程

```bash
ps -l
```

列出所有的进程

```bash
ps aux
# 显示的格式不一样
ps -lA
```

以树状形式列出所有用户的进程

```bash
ps axjf
pstree
```
