## Installation

```bash
npm i docsify-cli -g
```

## Commands

快速帮助

```bash
docsify -h
```

初始化项目

```bash
# 在当前目录
docsify init .
# 在 docs 目录
docsify init ./docs
```

本地预览 <http://localhost:3000>

```bash
docsify serve .
docsify serve docs
```
