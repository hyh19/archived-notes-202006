## Docker

<https://www.gitpod.io/docs/config-docker/>

```yml
image: node:alpine
```

```yml
image:
  file: .gitpod.Dockerfile
```

> Note: Currently, Gitpod only supports Debian/Ubuntu or Alpine based images.
