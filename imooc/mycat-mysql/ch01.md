# 第 1 章 课程介绍

## [1-1 MyCAT导学](https://coding.imooc.com/lesson/208.html#mid=13583)

MyCAT 是一个数据库中间件，其作用是：分库分表，提升整体性能。

其他数据库中间件

- OneProxy
- MaxScale

MyCAT 是基于阿里的 Cobar 开发的

[环境参数](https://coding.imooc.com/class/208.html#Envir)

- CentOS
- [MyCAT 1.6.5](http://dl.mycat.io/1.6.5/)
- JDK 1.7
- MySQL 5.7
