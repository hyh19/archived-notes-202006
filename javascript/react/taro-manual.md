## Install

<https://taro-docs.jd.com/taro/docs/GETTING-STARTED>

```bash
# 使用 npm 安装 CLI
npm install -g @tarojs/cli

# 或 使用 yarn 安装 CLI
yarn global add @tarojs/cli

# 或 使用 cnpm 安装 CLI
cnpm install -g @tarojs/cli
```

创建项目

```bash
taro init myapp

# 或 npm 5.2+ 也可在不全局安装的情况下使用 npx 创建项目

npx @tarojs/cli init myapp
```
