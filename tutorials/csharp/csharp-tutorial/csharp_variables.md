# [C# - Variables](https://www.tutorialspoint.com/csharp/csharp_variables.htm)

```c#
using System;

namespace VariableDefinition
{
    class Program
    {
        public static void Main(string[] args)
        {
            short a;
            int b;
            double c;

            a = 10;
            b = 20;
            c = a + b;
            Console.WriteLine("a = {0}, b = {1}, c = {2}", a, b, c);
            Console.ReadLine();
        }

    }
}
```
