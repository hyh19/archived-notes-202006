# [Swift - Variables](https://www.tutorialspoint.com/swift/swift_variables.htm)

```swift
// 不指定类型的变量必须初始化
var varA = 42
print(varA)

// 指定类型的变量可以不初始化
var varB:Float
varB = 3.14159
print(varB)
```

```swift
var varA = "Godzilla"
var varB = 1000.00
print("Value of \(varA) is more than \(varB) millions")
```
